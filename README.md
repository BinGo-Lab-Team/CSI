## 推荐方案

本项目基于 **Qt 6.9.2** 以及 **C++ 20** 构建。
如若使用低于此版本的 Qt，也最好不要低于 *Qt 6.8* 版本，建议使用 **Qt 6.8** 及以上版本。
推荐使用 **Visual Studio 2022** ，**Qt Creator** 或 **CLion 2025** 作为 IDE。
